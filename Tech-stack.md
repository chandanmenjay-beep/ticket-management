# Tech Stack

## Frontend
- **React with TypeScript**: For a robust, type-safe, and highly interactive ticket management dashboard.
- **Tailwind CSS**: For efficient and responsive UI styling.
- **React Router**: For client-side navigation within the single-page application.

## Backend
- **Node.js with Express and TypeScript**: A lightweight, flexible, and industry-standard framework for building fast RESTful APIs.
- **Database sessions**: For secure and reliable user authentication management.

## Database
- **PostgreSQL**: A powerful, open-source relational database to store tickets, user data, and system logs.

## ORM
- **Prisma**: For type-safe database access and a modern developer experience when interacting with PostgreSQL.

## AI Integration
- **Claude API (Anthropic)**: Leveraged for intelligent ticket classification, automated summarization, and generating suggested replies.

## Email Handling
- **SendGrid or Mailgun**: For reliable handling of outbound replies and processing incoming support tickets via webhooks.

## Deployment
- **Docker**: For consistent, containerized environments across development, staging, and production.
- **Cloud Provider (e.g., AWS, Railway, Fly.io)**: A scalable platform to host the containerized application and database.
